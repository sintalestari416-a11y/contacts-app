# 📱 Aplikasi Kontak (React Practice)

Selamat datang di proyek **Aplikasi Kontak** saya! Proyek ini adalah wadah latihan saya untuk memperdalam konsep *Stateful Component* dan *Controlled Component* di React.

## 📋 Tentang Proyek
Aplikasi ini memungkinkan pengguna untuk mengelola daftar kontak secara dinamis. Kamu bisa menambah kontak baru dengan nama dan tag, serta menghapus kontak yang sudah ada. Data awal diambil secara otomatis dari modul data yang sudah disiapkan.

## 🚀 Fitur Utama
* **Tambah Kontak**: Input data nama dan tag secara responsif menggunakan *state* di React.
* **Hapus Kontak**: Fitur hapus data yang langsung memperbarui tampilan secara *real-time*.
* **Daftar Dinamis**: Menampilkan data kontak dalam bentuk list yang rapi.

## 🛠️ Teknologi yang Digunakan
* **React**: Library utama untuk membangun antarmuka pengguna.
* **Vite**: Alat pengembangan (build tool) untuk pengalaman koding yang super cepat.
* **JavaScript (ES6+)**: Logika utama aplikasi, termasuk penggunaan *array filtering* dan *destructuring*.

## 📂 Struktur Penting
* `ContactApp.jsx`: *Root component* yang mengelola seluruh data dan fungsi utama.
* `ContactInput.jsx`: Komponen khusus untuk menangani form input pengguna.
* `data.js`: Sumber data awal untuk inisialisasi aplikasi.
